# Student Dropout and Success Prediction

Course: Machine Learning / Academic Project  
Type: Predictive Analytics and Classification Project  
Role: Data Preparation, Modeling, and Evaluation Contributor

## The Problem

Student outcomes such as dropout, graduation, and continued enrollment are influenced by multiple academic and demographic factors. This project was developed to analyze student data and build a machine learning model that can help identify patterns related to student dropout and academic success.

## Technical Stack

- **Programming Language:** Python
- **Notebook Environment:** Jupyter Notebook
- **Libraries:** Pandas, NumPy, Scikit-learn
- **Other Tools:** GitHub

## System Architecture & Logic

### 1. Data Preparation
The dataset was cleaned and prepared to make the information suitable for machine learning analysis.

### 2. Feature Analysis
Relevant student-related variables were examined to better understand which factors may influence academic outcomes.

### 3. Model Development
A machine learning model was developed using the notebook workflow to classify or predict student outcomes based on the dataset.

### 4. Performance Evaluation
The project includes model testing and evaluation to measure how effectively the model performs on the student dataset.

### 5. Insight Generation
The analysis supports a better understanding of dropout and success patterns, which can be useful for academic monitoring and intervention.

## Project Files

- `Students Dropout And Success.csv`
- `Students_Dropout_and_Success_MLP.ipynb`

## Project Outcome

This project strengthened my skills in data preparation, machine learning modeling, and evaluation using student data. It also improved my ability to apply predictive analytics to an educational dataset and turn model results into meaningful insights.
