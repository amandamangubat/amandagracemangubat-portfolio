# Metabase Pharmacy Inventory Dashboard System

Course: Database Management / Business Intelligence Project  
Type: Pharmacy Inventory, Reporting, and Dashboard System  
Role: Database Design, Reporting, Dashboard, and Website Integration Contributor

## The Problem

Pharmacies need a reliable way to manage stock, monitor sales, track expirations, and generate reports for better decision-making. This project was developed to create a pharmacy inventory system supported by a structured database, SQL-based reporting, website integration, and dashboard visualization through Metabase.

## Technical Stack

- **Database:** SQL
- **Dashboard Tool:** Metabase
- **Database Management Tool:** DBeaver
- **Website Component:** Integrated project website
- **Other Tools:** GitHub

## System Architecture & Logic

### 1. Inventory Data Management
The system stores and organizes pharmacy inventory records, making it easier to track product availability and stock movement.

### 2. Sales and Expiration Monitoring
The project includes logic for monitoring product sales and identifying items that may expire, helping improve inventory control.

### 3. SQL-Based Reporting
SQL scripts were created to generate reports and support more structured analysis of inventory-related data.

### 4. Dashboard Visualization
Metabase was used to transform the stored data into visual dashboards for clearer reporting and easier interpretation.

### 5. Website Integration
The project also includes a website component that supports the presentation and integration of the pharmacy inventory system.
