# Ciel d'Or: Luxury Restaurant UI/UX Design Project

Course: Hackathon / Design Project  
Type: Figma Prototype and Branding Design  
Role: UI, Layout, and Presentation Designer  
Live Prototype: [View Here](https://www.figma.com/proto/KCqvNMoVLsVIGrBbfuAS2x/Ciel-d-Or-Website?node-id=36-39&starting-point-node-id=36%3A39&t=mosDk0ulZuCh034z-1)

## The Problem

Luxury dining brands need a digital presence that reflects elegance, refinement, and clear communication. Ciel d'Or was developed as a Figma-based restaurant design project that combines branding, layout design, and polished presentation to create a premium online experience.

## Technical Stack

- **Design Tool:** Figma
- **Supporting Tools:** Canva, GitHub

## System Architecture & Logic

### 1. Luxury Brand Presentation
The design reflects a refined restaurant identity through elegant visuals and polished layout choices.

### 2. Organized Information Flow
Restaurant details are presented in a way that is easy to read and visually structured for users.

### 3. User-Friendly Navigation
Sections are arranged to guide users smoothly through the overall interface concept.

### 4. Branding-Focused Design
The project combines interface design and brand presentation to create a cohesive and memorable identity.

### 5. Prototype Development
The project was created as a Figma prototype to showcase the website concept, layout, and visual experience.

## Project Outcome

This project helped me strengthen my skills in visual presentation, interface layout, and branding-focused design. It also improved my ability to create a polished and user-friendly concept for a premium website experience.
